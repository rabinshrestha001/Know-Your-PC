  • Capacity (GB)

Central Processing Unit

The Brain

Why it matters

The CPU handles all instruction execution. A faster CPU means snappier system responsiveness and better performance in complex tasks like gaming or rendering.

Cores Physical processing units
Threads Virtual cores for multitasking
Clock Speed Cycles per second (GHz)

Pro Tip: More cores aren't always better for gaming. Look for high single-core performance (clock speed) for most games.

Graphics Processing Unit

The Visulaizer

Why it matters

Crucial for gaming, video editing, and 3D rendering. It takes the heavy lifting of visual calculations off the CPU.

VRAM Video Memory for textures
CUDA/Stream Processors Parallel processing units
TDP Power consumption (Watts)

Pro Tip: Check the VRAM requirements of the games you want to play. 8GB is the new minimum for 1080p gaming.

Random Access Memory

Short-term Memory

Why it matters

RAM stores data that the CPU needs immediately. More RAM allows you to run more applications simultaneously without slowing down.

Capacity Total size (8GB, 16GB, 32GB)
Speed Data transfer rate (MHz/MT/s)
Latency (CL) Response delay (Lower is better)

Pro Tip: Always use two sticks of RAM (Dual Channel) instead of one larger stick for significantly better performance.

Motherboard

The Foundation

Why it matters

It's the circuit board that connects all your hardware. It determines what CPU you can use, how much RAM you can add, and what ports you have.

Socket CPU physical interface (e.g. LGA1700, AM5)
Chipset Determines features (Z790, B650)
Form Factor Size (ATX, Micro-ATX, ITX)

Pro Tip: Ensure your motherboard has built-in Wi-Fi if you can't use an ethernet cable.

Storage (SSD/NVMe)

The Vault

Why it matters

Storage determines how many games, photos, and files you can keep. An NVMe SSD is crucial for fast boot times and loading screens.

Interface NVMe (Fastest) vs SATA (Slower)
Generation PCIe Gen 3, 4, or 5 (Speed tier)
Endurance (TBW) Total Bytes Written (Lifespan)

Pro Tip: Keep your OS on a fast NVMe SSD and use cheaper SATA SSDs or HDDs for bulk storage like movies.

Power Supply Unit

The Lifeline

Why it matters

A bad PSU can destroy your entire PC. A good one provides stable power and protects components from surges.

Efficiency 80+ Bronze, Gold, Platinum
Modularity Detachable cables for cable management
Headroom Extra wattage for upgrades

Pro Tip: Never cheap out on the PSU. Look for "80+ Gold" ratings from reputable brands like Corsair, Seasonic, or EVGA.
